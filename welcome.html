Welcome to Tabify

Tabify is a browser extension that helps you declutter your workspace and manage your tabs effectively.

Features:

  • Preventing duplicate tabs
  • Ability to ignore search parameters when considering tabs 'duplicate'
  • Ability to ignore anchor tags when considering tabs 'duplicate'
  • Workspace management

Get Involved

Tabify is an open-source project hosted on GitHub. You can find the source code on the extension's GitHub Repository.
